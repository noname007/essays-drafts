## 源码

```C
#include<signal.h>
#include<unistd.h>

static int sleepFlag = 1;

static void alrm_handler(int signo){
    sleepFlag = 0;
}

int mySleep(int sec)
{
    signal(SIGALRM,alrm_handler);
    alarm(sec);
    
    while(sleepFlag);
    
    return 0;
}
```


开启优化选项后，程序变成了死循环。从下面的汇编代码，不难发现编译器优化了 sleepFlag 的读取，导致了死循环。
